## Main Packages

| Package                                   | Purpose                                                       |
| ----------------------------------------- | ------------------------------------------------------------- |
| [Route](/docs/reference/route)            | Build powereful and atomic web-standard request handlers      |
| [Client](/docs/reference/client)          | Create type-safe clients for any REST API                     |
| [Router](/docs/reference/router)          | Match incoming `Request`s to request handlers                 |
| [OpenAPI Spec (OAS)](/docs/reference/oas) | Define, generate and infer OpenAPI schema without any codegen |
